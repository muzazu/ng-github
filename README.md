# Aplikasi dari Github API
Aplikasi menggunakan github open API dan Angular 1.75

## Penginstalan
1. Install dependency
```javascript
npm install
```
2. Jalankan Aplikasi

```javascript
npm start
```
3. Build

```javascript
npm build
```
